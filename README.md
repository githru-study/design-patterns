## 진행방식

- 개요 정리 + 사용사례(발췌 or 직접작성)
  - 개요 정리
    - 해결하고자 하는 문제 + 문제해결 아이디어 + 코드 (tobe only)

  - 사용사례 (asis → tobe)
    - 발췌 : 책/유튜브/블로그/github 등 다양한 소스에서 찾아보기 (개요와 연결할 사례를 찾는게 목적)
    - 직접작성 : 직접 문제사례 정의 + 패턴적용 개선 (찾기에 너무 애매한 내용일때)
